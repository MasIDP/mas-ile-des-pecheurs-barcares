# Mas sur l’Île des Pêcheurs au Barcarès

Projet de site gratuit pour GitHub Pages.

## Contenu

- `index.html` : structure du site
- `style.css` : design
- `script.js` : galerie, estimation tarif, e-mail de réservation
- `images/` : photos optimisées du logement

## Mise à jour sur GitHub

Pour remplacer l’ancien site :

1. Ouvre ton dépôt GitHub `mas-ile-des-pecheurs`
2. Supprime les anciens fichiers si besoin
3. Envoie :
   - `index.html`
   - `style.css`
   - `script.js`
   - le dossier `images` avec toutes les photos
4. Clique sur `Commit changes`
5. Attends 1 à 3 minutes

## À modifier plus tard

Dans `script.js`, tu peux modifier les tarifs :

```js
const PRICES = {
  low: 95,
  mid: 130,
  high: 175,
  cleaning: 60
};
```
